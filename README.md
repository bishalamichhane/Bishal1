Installation
Just, Clone this repository -

git clone --depth=1 https://github.com/htr-tech/zphisher.git
Now go to cloned directory and run zphisher.sh -

cd zphisher
bash zphisher.sh
pkg install tur-repo
pkg install zphisher
zphisher
